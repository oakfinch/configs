## Installation

Using `npm`:

```
npm install --save-dev @oakfinch/prettier-config
```

Using `yarn`:

```
yarn add -D @oakfinch/prettier-config
```

## Usage

`.prettierrc.js`

```
module.exports = {
  ...require("@oakfinch/prettier-config"),
  // your rules here
};
```
