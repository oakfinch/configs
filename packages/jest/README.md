## Installation

Using `npm`:

```
npm install -D @oakfinch/jest-config
```

Using `yarn`:

```
yarn add -D @oakfinch/jest-config
```

## Usage

`jest.config.js`
```
module.exports = {
  preset: '@oakfinch/jest-config',
};
```
