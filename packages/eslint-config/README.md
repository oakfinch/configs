# eslint-config
Base ESLint config for oakfinch projects

------

[github](https://github.com/oakfinch/eslint-config) | [npm](https://www.npmjs.com/package/@oakfinch/eslint-config)

------

## Installation

Using `npm`:

```
npm install --save-dev @oakfinch/eslint-config
```

Using `yarn`:

```
yarn add -D @oakfinch/eslint-config
```

## Usage

`.eslintrc.json`
```
{
    "extends": "@oakfinch/eslint-config"
}
```
