## Installation

Using `npm`:

```
npm install -D @oakfinch/eslint-config
```

Using `yarn`:

```
yarn add -D @oakfinch/eslint-config
```

## Usage

`.eslintrc.json`
```
{
    "extends": "@oakfinch/eslint-config"
}
```
